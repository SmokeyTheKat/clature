## Build
make targets
| Target | Description |
|-------:|-------------|
| _(none)_ | complies to ./ddScript |
| Check | runs test scripts in ./test/ |
| install | compiles to /usr/bin/cds and adds config files to /usr/share/cds/ |
| unstall | removes config files and file from /usr/bin |
